jinka クラスファイル ver. 1.1

インストール

手順の概要

jinka.cls は jinka.dtx というファイルから LaTeX の DOCSTRIP ユーティリティを用いて生成します．この作業を簡略にするため，.ins ファイルを用意しました．インストールに必要な作業は次の2点です．

  1. LaTeX で jinka.ins ファイルを処理し，jinka.cls を生成する．
  2. 必要なファイルを適当なディレクトリにコピーする．

「適当なディレクトリ」とは，TeX の検索パスに含まれるディレクトリのこと です．（たとえば，文書を作成し LaTeX のコンパイルを実行するディレクトリ）

UNIX 系の環境での実行例

インストールするディレクトリを ${HOME}/lib/tex とすると，具体的には 次のように実行することになります．

$ platex jinka.ins
$ cp jinka.cls ${HOME}/lib/tex
$ cp jjpsy.bst ${HOME}/lib/tex

make がインストールされている環境では，次の方法でインストール することもできます． ただし，Makefile に変更を加える必要があるかもしれません．

$ make all
$ make install TEXDIR=${HOME}/lib/tex
