jinka クラスファイル ver. 1.1

jinka.cls は，大阪大学人間科学 研究科/学部 の 修士論文/卒業論文 を作成 するための LaTeX2e クラスファイルです．

INSTALL

手順の概要

jinka.cls は jinka.dtx というファイルから DOCSTRIP ユーティリティを 用いて生成します．インストール作業を簡略にするためのバッチファイルを用意 しました．インストールに必要な作業は次の2点です．

  1. LaTeX で jinka.ins ファイルを処理し，jinka.cls を生成する．
  2. 必要なファイルを適当なディレクトリにコピーする．

「適当なディレクトリ」とは，TeX の検索パスに含まれるディレクトリの ことです．

UN*X 系の環境での実行例

インストールするディレクトリを ${HOME}/lib/tex とすると，具体的には 次のようなコマンドを実行することになります．

$ platex jinka.ins
$ cp jinka.cls ${HOME}/lib/tex
$ cp jpa.bst ${HOME}/lib/tex

make がインストールされている環境では，次の方法でインストール することもできます．ただし，Makefile に変更を加える必要があるかもしれません．

$ make all
$ make install TEXDIR=${HOME}/lib/tex

Macintosh や Windows 版での注意点

クラスファイルの漢字コードは EUC，改行コードは LF となっています． Macintosh や Windows で使用する際には，漢字コードを Shift JISに，改行コードを CR (Mac) もしくは CR+LF (Win) に変換する必要があるかもしれません．
